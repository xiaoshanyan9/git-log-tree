**git-log-tree** is a fancy `git log --graph`.

![](sample.svg)

## Installation

```sh
curl -L -o git-log-tree https://github.com/xiaoshanyan9/git-log-tree/raw/refs/heads/main/git-log-tree
chmod +x git-log-tree
sudo mv git-log-tree /usr/local/bin/
```

## Usage

```sh
git log-tree
```
